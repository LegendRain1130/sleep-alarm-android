# Sleep Alarm Android

一个 Android 夜间睡眠闹铃 App。

## 功能
- 夜间定时启动监测
- 熄屏后自动按设定睡眠时长预定闹铃
- 亮屏自动取消并等待下一次熄屏
- 支持自定义铃声
- 支持夜间开始/结束时间配置

## 当前状态
- 本体闹铃链路已可用
- 当前版本以测试为主

## 目标方向和已知问题
- 增加加速度传感器和充电状态识别
- 接入穿戴设备的睡眠监测功能
- 确认或修复由于手机通知/来电导致的亮屏导致的倒计时重置(已修复)

## 注意事项
- 不同厂商系统的省电策略可能影响后台监测和闹铃表现
- 建议将应用加入电池优化白名单或设为“不限制”
